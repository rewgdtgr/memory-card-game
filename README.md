# 🎴 记忆翻牌游戏

一款基于微信小程序开发的记忆翻牌小游戏，锻炼你的记忆力，挑战最高分数！

## 📱 项目简介

这是一个微信小程序项目，实现了经典的记忆翻牌游戏。玩家需要翻开卡片，找到相同图案的配对，在最短时间内完成所有配对即可获胜。

## ✨ 功能特点

- 🎮 **三种难度模式**：简单(4×3)、中等(4×4)、困难(6×4)
- ⏱️ **计时功能**：记录完成游戏所用时间
- 🏆 **计分系统**：根据用时和步数计算得分
- 💡 **提示功能**：卡住时可以使用提示
- 📊 **本地存储**：保存最高分、游戏次数和最佳时间
- 🎨 **精美UI**：渐变背景、卡片翻转动画、配对成功特效

## 🎯 游戏规则

1. 点击卡片翻转查看图案
2. 每次翻开两张卡片进行配对
3. 两张相同图案的卡片配对成功
4. 找出所有配对即可获胜
